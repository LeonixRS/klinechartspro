KLineChart Pro - Minimal Example

A simple demonstration of klinecharts/pro with sample financial data

Note: This example uses sample mock data for demonstration. For real trading data, you would need to implement a proper datafeed or use the DefaultDatafeed with a Polygon.io API key.

About this example:

  • Uses the latest stable version of @klinecharts/pro (v0.1.1)
  • Loads via CDN for simplicity
  • Demonstrates basic chart initialization with sample data
  • Shows a simple custom datafeed implementation
