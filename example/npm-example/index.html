KLineChart Pro - NPM Example

A simple demonstration using npm installation and module bundler

Note: This example uses sample mock data for demonstration. For real trading data, you would need to implement a proper datafeed or use the DefaultDatafeed with a Polygon.io API key.

About this example:

  • Uses npm installation of @klinecharts/pro (v0.1.1)
  • Built with Vite for development and bundling
  • ES6 modules and modern JavaScript
  • Shows TypeScript-style imports

Commands:

  • npm install - Install dependencies
  • npm run dev - Start development server
  • npm run build - Build for production
