KLineChart Pro - Minimal Example (CDN Version)

About This Example

This is a minimal working example of @klinecharts/pro displaying a candlestick chart with the DefaultDatafeed (using CDN).

This version uses the published library from CDN to test compatibility.

Features demonstrated:

  • Basic chart initialization
  • DefaultDatafeed with demo API key
  • Symbol information display
  • Time period configuration
  • Volume chart display

Note: This example uses the DefaultDatafeed with a demo API key, so it may not load real data.
