KLineChart Pro - Minimal Example

About This Example

This is a minimal working example of @klinecharts/pro displaying a candlestick chart with dummy data.

The chart shows sample OHLCV (Open, High, Low, Close, Volume) data for demonstration purposes.

Features demonstrated:

  • Basic chart initialization
  • Custom datafeed implementation with static data
  • Symbol information display
  • Time period configuration
  • Volume chart display
